# 地震ビジュアライザー

USGS Earthquake Catalog APIから実際の地震データを取得し、地図上にマグニチュードに応じたエフェクトで可視化する作品です。

## 機能

### データ取得
- USGS Earthquake Catalog API からM5.0以上の地震データを年単位で取得。

### ビジュアルエフェクト
地震のマグニチュードに応じて以下のエフェクトが変化します（内部値1〜8にマッピング）。

| エフェクト | 説明 | 発動条件 |
|---|---|---|
| 波紋（Ripple） | 震源から広がる同心円。多重描画による発光表現 | 全地震 |
| パーティクル（Particle） | 震源から飛散する破片 | 全地震 |
| 衝撃フラッシュ（Flash） | 震源の放射状グロー＋白い閃光 | M5.0以上 |
| 亀裂（Crack） | 放射状に伸びるジグザグ線 | M6.0以上 |
| 画面揺れ（Screen Shake） | 画面全体の振動 | 全地震（強度に比例） |

### 色表現
マグニチュードに応じたグラデーションで表示されます。
- 弱（M4〜5）: 青系
- 中（M5〜7）: 黄系
- 強（M7〜9）: 赤系

## 操作方法

| キー | 操作 |
|---|---|
| `←` / `→` | 表示年を切り替え |
| `SPACE` | 現在の年をリプレイ（自動再生） |
| マウスクリック | 個別の地震エフェクトを手動で発生させる |
| マウスホバー | マグニチュードと発生場所のツールチップを表示 |

## 動作要件

- Processing 4.x
- インターネット接続

## 使用API・データソース

- [USGS Earthquake Catalog API](https://earthquake.usgs.gov/fdsnws/event/1/)
- [Natural Earth 110m Land GeoJSON](https://github.com/nvkelso/natural-earth-vector)
